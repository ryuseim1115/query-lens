# 業務フロー

```mermaid
graph TD
    %% ユーザーの操作
    Start((開始)) --> DBAccess[データベースへログインする]
    DBAccess --> InputSQL[SQL入力欄にSQLを記述]

    %% システムの自動処理
    InputSQL --> ParseResult

    subgraph システム処理
        ParseResult{SQL解析成功?}
        ParseResult -->|失敗| ParseError[エラー表示]
        ParseResult -->|成功| DetectSelect{SELECT文?}
        DetectSelect -->|いいえ| SelectError[エラー表示]
        DetectSelect -->|はい| ExtractSubqueries[サブクエリを取得]
        ExtractSubqueries --> DetectTables[サブクエリごとのテーブルを抽出]
        DetectTables --> DisplayTables[サブクエリごとのテーブルをすべて画面に表示]
    end

    %% インタラクション
    DisplayTables --> ClickSQL[SQL入力欄のサブクエリをクリック]

    ClickSQL --> Highlight

    subgraph システム処理2[クリック応答処理]
        Highlight[クリックされたサブクエリに関連するテーブルを着色]
        Highlight --> Preview[クリックされたサブクエリの結果（データ）をプレビュー表示]
    end
